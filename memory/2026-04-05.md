## ClawPilot + PocketClaw Setup

### What happened
熊老大想用 PocketClaw 连接 OpenClaw gateway。走了不少弯路。

### Steps that worked
1. `npm install -g @rethinkingstudio/clawpilot@latest --prefix ~/.npm-global` (因为 /usr/local 权限不够)
2. `export PATH="$HOME/.npm-global/bin:$PATH"` 加到 ~/.bashrc
3. `clawpilot install` — 安装 pm2 服务
4. `clawpilot set-token <token>` — 解决 "gateway token missing" 错误，token 从 ~/.openclaw/openclaw.json 里找
5. `clawpilot restart` — 让 token 生效
6. `clawpilot pair` — 生成配对码 XACN-E7KS
7. PocketClaw app 扫码配对

### Key troubleshooting
- **Problem**: ClawPilot 连 gateway 报 `unauthorized: gateway token missing`
- **Fix**: `clawpilot set-token` 把 openclaw.json 里的 token 写入 ~/.clawai/config.json
- **Problem**: `openclaw gateway restart` 后 ClawPilot 进程消失
- **Root cause**: pm2 systemd 服务 (`pm2-xionghouyuan2.service`) 是 enabled 但 inactive (dead)，机器重启后没有自动复活
- **Fix**: 
  1. `sudo systemctl start pm2-xionghouyuan2` — 让 systemd 启动 pm2 守护
  2. pm2 会自动从 `~/.pm2/dump.pm2` 恢复 ClawPilot
  3. 之前已执行过 `pm2 save`，进程列表已保存

### Final status

- ClawPilot: online, Relay ✅ connected, Gateway ✅ connected
