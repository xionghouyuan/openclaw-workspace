## html2img 工具

**脚本：** `~/.openclaw/workspace/scripts/html2img.js`
**安装：** `npm install puppeteer-core`（用已有的 Chrome）
**使用：**
```bash
node scripts/html2img.js <输入.html> <输出.jpg>
```

**功能：**
- puppeteer-core + Chrome headless
- 自动获取页面高度（fullPage: true）
- 输出 JPEG，质量 92%
- 默认路径：/usr/bin/google-chrome
